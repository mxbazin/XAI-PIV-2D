# Data (`data/`)

This folder is **not version-controlled** (see `.gitignore`): the full dataset
is ~13 GB, which exceeds GitHub's limits. The data must be placed here manually
(or hosted on Zenodo / external storage and downloaded before running).

## Expected structure

Paths and file patterns are defined in
[`code/configuration/folders.py`](../code/configuration/folders.py). The token
`$INDEX$` is the field index and `$NSAMPLES$` the number of SHAP samples.

```text
data/
├── piv/                       # raw PIV velocity fields    (piv.$INDEX$.h5.uvw)
│   └── phase_metadata.npy     # cycle phase metadata
├── tfrecord/                  # generated TFRecords (prepare_tfrecords.py)
├── SHAP/                      # per-point SHAP values       (piv_jets_nsample$NSAMPLES$.$INDEX$.h5.shap)
├── SHAPq/                     # percolated SHAP structures  (piv_jets_nsample$NSAMPLES$.$INDEX$.struc)
├── Q/                         # Q events (Reynolds stress)  (piv_jets.$INDEX$.Q)
├── Streak/                    # streaks                     (piv_jets.$INDEX$.Lstreaks)
└── Vortices/                  # Chong vortices              (piv_jets.$INDEX$.Chong)
```

## Reproduction

Only the `piv/` fields are required to start from scratch: the `SHAP/`,
`SHAPq/`, `Q/`, `Streak/` and `Vortices/` folders are regenerated by the
pipeline described in the [main README](../README.md).
